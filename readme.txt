Add image over image which changes over hover with animations.


== Description ==

Image Over Image is a <a href="http://codecanyon.net/item/visual-composer-page-builder-for-wordpress/242431?ref=labibahmed">Visual Composer Extension</a> which will allow you to add image over image which changes image when you hover over a image. It contains pure CSS3 animations. You can also add link to the images.It is a multipurpose plugin you can use it to display your <b>products,banners,portfolio,team</b> etc.

<b>Note:</b> This requires <a href="http://codecanyon.net/item/visual-composer-page-builder-for-wordpress/242431?ref=labibahmed">Visual Composer Plugin</a>
